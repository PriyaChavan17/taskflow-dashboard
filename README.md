TaskFlow Dashboard

A modern productivity dashboard web application to manage weekly tasks, track progress, and visualize performance using interactive charts.

1. Problem Statement

Managing daily and weekly tasks efficiently is challenging without proper tracking and visualization. Users often lack insights into their productivity, progress, and completion rates.

This project solves the problem by providing:
- A structured weekly planner
- Visual analytics of task completion
- Goal tracking system


 2.Prompt Used

“Build a professional dashboard web app with login system, weekly planner, multiple charts (line, bar, pie), goal tracking, and modern UI using HTML, CSS, and JavaScript.”



3.Architecture

Frontend Architecture:

- Login Page (index.html)
  - User authentication using LocalStorage

- Dashboard Page (dashboard.html)
  - Task management UI
  - Charts visualization
  - Goal tracking

- Styling (style.css)
  - Responsive design
  - Dark mode support

- **Logic (script.js)**
  - Task CRUD operations
  - LocalStorage data handling
  - Chart rendering using Chart.js

Flow:
User Login → Dashboard → Add Tasks → Data Stored → Charts Update


 4. Development Steps

5. Features

- Login system (LocalStorage-based)
-  Weekly task planner
-  Line chart (completed tasks trend)
-  Bar chart (daily tasks count)
-  Pie chart (completed vs pending with %)
-  Weekly goal tracking
-  Dark mode toggle
-  Responsive design
-  Persistent data using LocalStorage
